How to Use the Data Importer

1. Select Input File: Choose your CSV or Excel file. Content is previewed on the right.

2. Select Output Sheet: Choose an existing sheet or create a new one. "Append" adds to the bottom, while "Overwrite" clears the sheet first.

3. Trim Data: Used to remove unwanted headers, footers, or margins by skipping rows/columns.

4. Find & Replace: Define rules to clean data. Use exact mode to replace only cells whose entire value matches (e.g. "N/A" → blank), or contains mode to replace all occurrences of the text within each cell (e.g. strip "$" from "$1,200").

Settings are automatically saved and re-applied next time you select the same Output Sheet!
